## Executar um arquivo .js

  `node src/index.js`
  ou
  `node --watch src/index.js`
  para monitorar as alterações

## HTTPie

  ### Instalar o HTTPie
  ```	bash
  sudo apt update
  sudo apt install httpie
  ```

  ### Executar o HTTPie
  ```	bash
  http GET http://localhost:3333
  ```

## Executar um script

  ```	bash
  npm run dev
  ```
